# Student-management-system
A Student Management System (SMS) developed using Python in a Jupyter Notebook is a simple application designed to store, manage, and process student-related data efficiently. It provides an easy way to handle records such as student ID, name, marks, attendance, and other academic details.
The system aims to reduce manual record-keeping by providing an automated way to store, update, search, and delete student records. It works using basic Python concepts like lists, dictionaries, and file handling, allowing data to be stored either temporarily in memory or permanently in files such as text or CSV. The use of Jupyter Notebook makes the system interactive and easy to understand, as the program can be written and executed step-by-step in separate cells. This project is especially useful for beginners as it helps in understanding core programming concepts like input/output operations, conditional statements, and data manipulation. Although it is limited to small-scale applications and lacks advanced features like a graphical user interface or strong security, it serves as a strong foundation for developing more complex management systems in the future.
Technologies Used
Python (core programming)
Jupyter Notebook (interactive coding environment)
Optional libraries:
pandas (for data handling)
os (for file management)
Key Features:
➤ Add Student Records
Users can enter details like ID, name, and marks.
➤ Display Records
Shows all stored student information in a structured format.
➤ Search Student
Find a student using ID or name.
➤ Update Details
Modify existing student records.
➤ Delete Records
Remove a student from the system.
➤ Calculate Results
Compute total marks, average, and grades.
